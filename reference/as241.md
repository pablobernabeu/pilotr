# Inverse normal cumulative distribution function (Wichura's AS 241)

Compute the standard-normal quantile for a probability using Wichura's
(1988) Algorithm AS 241 (the PPND16 routine), the same algorithm
underlying [`stats::qnorm()`](https://rdrr.io/r/stats/Normal.html).

## Usage

``` r
as241(p)
```

## Arguments

- p:

  A probability in the open interval (0, 1).

## Value

The standard-normal quantile (a numeric value) corresponding to `p`.

## Examples

``` r
as241(0.975)   # about 1.959964
#> [1] 1.959964
```
