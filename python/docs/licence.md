# Licence

pilotr is released under the MIT licence, reproduced below from the
repository's [`LICENSE`](https://github.com/pablobernabeu/pilotr/blob/main/python/LICENSE)
file.

```python exec="1" html="1"
import html

with open("LICENSE", encoding="utf-8") as f:
    text = f.read()

# The <code> child is what the stylesheet's overflow guard keys on, so the licence's
# long lines scroll inside the block rather than stretching the page.
print("<pre><code>" + html.escape(text) + "</code></pre>")
```
